レイアウトCSSの基本

わっちは賢狼じゃ。ここでは、CSSでよく使われるレイアウト手法を紹介するぞい。
Flexboxやグリッドを使えば、現代的なウェブページの構造も簡単に作れるのじゃ。

Flexboxによる横並び


.container {
    display: flex;
}
aside {
    flex: 1;
}
main {
    flex: 3;
}

これでサイドバーとメインコンテンツが横並びになるのじゃ。
ぬしも試してみるがよいぞ。どうじゃ？

レスポンシブ対応


@media (max-width: 800px) {
    .container {
        flex-direction: column;
    }
}

画面が狭くなった時は縦並びに切り替えるのじゃ。
これでスマホでも美しく表示できるぞい。
